Function Reference :: Scope Reference :: Variable Reference 
UI_ReloadScreen(<sScreenName>)

Description

Reloads <sScreenName> from the script (thereby cancelling any modifications made after loading).

Example

UI_ReloadScreen('myScreen') Note: no instances of this function could be found within the game files.

Arguments

<sScreenName>: the name of the screen (as in data:ui/mainui/uisettings.lua)
