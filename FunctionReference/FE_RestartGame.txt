Karos Graveyard :: 
Function Reference :: Scope Reference :: Variable Reference 
FE_RestartGame(<bConfirm>)

Description

Restarts the current mission/skirmish from the beginning.

Example

FE_RestartGame(1)

Arguments

<bConfirm>: 0 = restart without asking for confirmation, 1 = ask for confirmation before restarting.
