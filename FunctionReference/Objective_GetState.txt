Objective_GetState(<iObjectiveIndex>)

Description

Returns the current state of an objective as an index number. The index numbers correspond to the OS_Off, OS_Incomplete, OS_Complete, and OS_Failed variables.

Example


Arguments

<iObjectiveIndex>: the index number of the objective (returned by Objective_Add).

Gamerule
