Function Reference :: Scope Reference :: Variable Reference 
SobGroup_SetMadState(<sSobGroupName>, <sStateName>)

Description

Makes all ships in <sSobGroupName> play the named animation.

Example

SobGroup_SetMadState("Droid_1", "Docked")

Arguments

<sSobGroupName>: name of the sobgroup
<sStateName>: name of the animation as defined in the ship .madstate file
