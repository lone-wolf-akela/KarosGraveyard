UI_SetElementEnabled(<sScreenName>, <sButtonName>, <bEnable>)

Description

Enables/disables a button of a UI screen.

Example

UI_SetElementEnabled("NewLaunchMenu", "launchButton", 0)

Arguments

<sScreenName>: the name of the screen.
<sButtonName>: the name of the button or screen element; the name must be given in the appropriate UI file.
<bEnable>: 0 = disable, 1 = enable.
