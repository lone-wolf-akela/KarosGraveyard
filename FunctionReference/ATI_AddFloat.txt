ATI_AddFloat(<iATIIndex>, <fParameter>)

Description

Adds a float parameter to an ATI display item.

Example

ATI_AddFloat(RRPI_PlayerProgress, progress)

Arguments

<iATIIndex>: the index number of the parameter.
<fParameter>: the float parameter.
