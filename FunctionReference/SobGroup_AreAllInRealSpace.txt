SobGroup_AreAllInRealSpace(<sSobGroupName>)

Description

Returns true if every ship in <sSobGroupName> is in real space (may be docked or not).

Example

if SobGroup_AreAllInRealSpace("Keeper")==1 then ... code ... end

Arguments

<sSobGroupName>: the name of the sobgroup.
