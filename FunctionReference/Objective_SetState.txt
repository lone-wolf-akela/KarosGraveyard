Objective_SetState(<iObjectiveIndex>, <iState>)

Description

Sets the state of an objective.

Example


Arguments

<iObjectiveIndex>: the index number of the objective (returned by Objective_Add).
<iState>: the index number of the state. The index numbers correspond to the OS_Off, OS_Incomplete, OS_Complete and OS_Failed variables.

Scope

Gamerule
