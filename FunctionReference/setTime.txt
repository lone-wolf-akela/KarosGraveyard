setTime(<sShipName>, <sAnimName>, <fTime>)

Description

Sets the current time in an animation (FF or Rew)
Returns nothing

Example


Arguments

<sShipName>: Sob* Passed from game
<sAnimName>: string Name of mesh animation from ".mad" file.
<fTime>: float Time (seconds) in animation to jump to. Will be clamped to animation bounds.
