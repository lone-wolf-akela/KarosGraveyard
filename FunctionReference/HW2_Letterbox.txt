HW2_LetterBox(<bSwitch)

Description:

This function must be called from within an event table. It enables/disables the letterbox mode (usually appears when speech is spoken). Use Camera_SetLetterboxStateNoUI for switching between letterbox modes from a normal chunk.
Note: this function is defined in "data\scripts\scar\scar_util.lua".

Arguments:

<bSwitch>: 0 = disable, 1 = enable.

Scope

Gamerule
