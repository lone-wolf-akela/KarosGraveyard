NISSetSkippable(<bSkippable>)

Description

Makes it so that any NISs currently active can be skipped using the escape key.

Example

Note: no instances of this function being used within the game files can be found.

Arguments

<bSkippable>: 0 = not skippable, 1 = skippable.
