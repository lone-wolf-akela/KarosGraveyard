setGlareIntensity (<fIntensity>)

Description

Changes the global value of the glare.
Note: setting glare above 0 doesn't work with all graphics chipsets/drivers. See this thread.

Example

setGlareIntensity(0.10)

Arguments

<fIntensity>: a float value between 0 and 1.
