Camera_FocusVolumeWithBuffer(<sVolumeName>, <fDistance>, <fBufferDistance>, <fTime>)

Description

Focus on the center of the volume. Any volume can be used.
If ZoomClose is set to 1, it will zoom in close to the bounding sphere of the volume.
If a time and distance is given, the ZoomClose flag will be set to false.
