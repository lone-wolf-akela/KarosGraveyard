Camera_AltFocusWithVolume(<sSobGroupName>, <bCloseZoom>, <sVolumeName>)

Description

Same as Camera_AltFocus, except a volume can be added to the focus.

Example


Arguments

<sSobGroupName>: the sobGroup to focus on.
<bCloseZoom>: will zoom in close if set to 1.
<sVolumeName>: the name of the volume that can be added to the focus selection.
