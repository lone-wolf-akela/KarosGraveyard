UI_SetPreviousScreen(<sCurrentScreenName>, <sPreviousScreenName>)

Description

Sets (preloads?) the previous screen before loading it using UI_PreviousScreen().

Example

UI_SetPreviousScreen("UserProfile", "NewMainMenu")

Arguments

<sCurrentScreenName>: the name of the current screen.
<sPreviousScreenName>: the name of the screen to load previously.
