ATI_AddString(<iATIIndex>, <sParameter>)

Description

Adds a string parameter to an ATI display item.

Example

ATI_AddString(0, "$8010")

Arguments

<iATIIndex>: the index number of the parameter.
<sParameter>: a localization ID or a raw string.
