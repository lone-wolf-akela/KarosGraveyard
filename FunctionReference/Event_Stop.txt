Event_Stop(sEventName)

Description

Stops an 'event' started with Event_Start.
Mainly used in tutorial.

Example

Event_Stop("intelevent_intro_taskbar_active")

Arguments

<sEventName> string with the name of the event defined as Events.sEventName in the Events table

Scope

Gametype
