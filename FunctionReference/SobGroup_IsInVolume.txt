SobGroup_IsInVolume(<sSobGroupName>, <sVolumeName>)

Description

Returns true if <sSobGroupName> is colliding with <sVolumeName>.
Note: Sleeping ships, such as those docked within a ship in the Volume, will return true -Grey_Ghost

Example


Arguments

<sSobGroupName>: the name of the sobgroup.
<sVolumeName>: the name of the volume.
