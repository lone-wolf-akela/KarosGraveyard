getPauseTime(<sShipName>, <sAnimName>)

Description

Returns the time when the specified animation will pause. A very large number means it will not pause.

Example


Arguments

<sShipName>: Sob* Passed from game
<sAnimName>: the name of the mesh animation from the ".mad" file.
