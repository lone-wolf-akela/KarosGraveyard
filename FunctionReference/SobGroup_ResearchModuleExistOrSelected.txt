SobGroup_ResearchModuleExistOrSelected(<sSobGroupName>)

Description

Returns an integer if a Research Module exists on any ship in <sSobGroupName>, and is selected.
Returns 1 if RM exists but is not selected, 2 if RM exists and is selected, 0 if RM does not exist, -1 on error.
