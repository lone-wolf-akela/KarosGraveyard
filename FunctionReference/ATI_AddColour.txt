ATI_AddColour(<iATIIndex>, <tParameter>)

Description

Add a colour parameter.

Example

ATI_AddColour(HCPI_PlayerColour, {1,1,1,1,})

Arguments

<iATIIndex>: the index number of the parameter.
<tParameter>: a table containing RGBA color components.
