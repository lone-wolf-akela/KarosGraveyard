Player_SetDefaultShipTactic(<iPlayerIndex>, <iTactic>)

Description

Sets the default tactics for all ships in the player's fleet.

Example

Player_SetDefaultShipTactic(1, AggressiveTactics)

Arguments

<iPlayerIndex>: the index number of the player.
<iTactic>: the index number of the tactic. Valid tactics are: AggressiveTactics, DefensiveTactics and PassiveTactics.
Note: the above tactics are the same as entering an index number.
