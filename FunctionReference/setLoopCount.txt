setLoopCount(<sShipName>, <sAnimName>, <nLoops>)

Description

Sets the number of times the animation will loop. 0 by default.
Returns nothing

Example


Arguments

<sShipName>: Sob* Passed from game
<sAnimName>: string Name of mesh animation from ".mad" file.
<nLoops>: int Number of times the animation will loop animations with no looping.
