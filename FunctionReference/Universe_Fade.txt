Karos Graveyard :: 
Function Reference :: Scope Reference :: Variable Reference 
Universe_Fade(<fPercentage>, <fTime>)

Description

Fades the screen over the given amount of time.

Example

Universe_Fade( 1, 2 )

Arguments

<fPercentage>: must be between 0 and 1. (0 = everything visible, 1 = black screen).
<fTime>: the amount of time in seconds it takes to complete the fade.
