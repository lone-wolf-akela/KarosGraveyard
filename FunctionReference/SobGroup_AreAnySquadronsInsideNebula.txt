SobGroup_AreAnySquadronsInsideNebula(<sSobGroupName>, <sNebulaName>)

Description

Returns true if any squadron from <sSobGroupName> is inside <sNebulaName>.

Example

if SobGroup_AreAnySquadronsInsideNebula("Player_Ships0", "Nebula_"..i)==1 then ...code... end

Arguments

<sSobGroupName>: the name of the sobgroup.
<sNebulaName>: the name of the nebula as defined in the .level file (see addNebula).
