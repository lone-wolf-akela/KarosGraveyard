NISPlayFromGroup(<sNISPath>, <sSobGroupName>)

Description

Unknown. Presumably plays the given NIS from the point of view of the named sobgroup.

Example

NISPlayFromGroup("nis/NIS02B", "Convoy_4")

Arguments

<sNISPath>: the path of the NIS file, relative to the "Data: directory, and minus the file extension.
<sSobGroupName>: the name of the sobgroup.
