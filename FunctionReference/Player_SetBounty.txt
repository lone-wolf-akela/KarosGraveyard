Player_SetBounty(<iPlayerIndex>, <iMultiplier>)

Description

Sets the bounty multiplier for <iPlayerIndex>.
Note: this function only applies within the Homeworld Classic mod by B1FF.

Example


Arguments

<iPlayerIndex>: the index number of the player.
<iMultiplier>: the bounty multiplier. An <iMultiplier> value of 1 will award a bounty equal to the build cost of any ships killed.
