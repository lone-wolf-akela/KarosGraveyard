SobGroup_SalvagedByPlayer(<sSobGroupName>, <iPlayerID>)

Description

Returns true if <sSobGroupName> is owned by the player and is sleeping.

Example

if SobGroup_SalvagedByPlayer("Derelict_Mover", 0)==1 then ...code... end

Arguments

<sSobGroupName>: name of the salvaged sobgroup
<iPlayerID>: player index number
