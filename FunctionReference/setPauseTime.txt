setPauseTime(<sShipName>, <sAnimName>, <fTime>)

Description

Specifies when the indicated animation will pause.
Returns nothing

Example


Arguments

<sShipName>: Sob* Passed from game
<sAnimName>: string Name of mesh animation from ".mad" file.
<fTime>: float Time (seconds) in animation when animation will pause. Pass a very large number to pause at the end.

Scope

Madstate
