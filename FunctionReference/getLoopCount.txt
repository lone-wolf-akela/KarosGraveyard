getLevelOfDifficulty()

Description

Returns the current level of difficulty (LOD) as an integer ranging between 0 and 3. 0 is "Easy", 1 is "Standard", 2 is "Hard" and 3 is "Expert".
The level of difficulty can be set using the setLevelOfDifficulty function.

Example

getLevelOfDifficulty()

Arguments

None.
