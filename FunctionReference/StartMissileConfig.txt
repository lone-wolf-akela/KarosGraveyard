StartMissileConfig()

Description

Instantiates and returns a Ship object to build a missile or mine around. This is always the first non-commented line in a .miss file.
Create a Ship for missile construction.
Returns a new Ship object.

Example

NewShipType = StartMissileConfig()

Arguments

None.
