setLightSpecular (<sLightName>, <tSpecular>)

Description

Changes the specular parameter of the name light.

Example


Arguments

<sLightName>: the name of the light.
<tSpecular>: a table containing the RGB components of the specular effect. Values greater than 1.0 are assumed to be byte values [0..255], and get scaled. So, we get a vector instead.
