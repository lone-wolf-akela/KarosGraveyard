Karos Graveyard :: 
Function Reference :: Scope Reference :: Variable Reference 
SobGroup_LoadPersistantData(<sShipType>)

Description

Causes the persistant fleet to spawn at "PlayerStartPoint" in formation around [...presumably the first occurance of...] <sShipType>.

Example

SobGroup_LoadPersistantData("Hgn_Mothership")

Arguments

<sShipType>: the type of ship within the persistant fleet to serve as the center of the parade formation.
