fogSetColour(<tColour>)

Description

Sets the colour and opacity for fog.

Example

fogSetColour(0.083984, 0.123047, 0.207031, 1)

Arguments

<tColour>: a table containing the RGBA components of the fog, where red, green and blue take values between 0 and 255, and alpha takes values between 0 and 1.
