Objective_Add(<sObjectiveName>, <iType>)

Description

Adds a new objective. Returns the index number of the objective.

Example


Arguments

<sObjectiveName>: the name of the objective. This can be a raw string or localization ID.
<iType>: index number of the type of objective. Can use OT_Primary or OT_Secondary instead.

Scope

Gamerule
