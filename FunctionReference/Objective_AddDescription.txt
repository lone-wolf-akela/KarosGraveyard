Objective_AddDescription(<iObjectiveIndex>, <sDescription>)

Adds a description to an objective.

Example


Arguments

<iObjectiveIndex>: the index number of the objective (returned by Objective_Add).
<sDescription>: the text description of the objective. Can be a raw string or localization ID.
