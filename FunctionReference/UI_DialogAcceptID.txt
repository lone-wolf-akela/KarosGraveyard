UI_DialogAcceptID(<iScreenIndex>)

Description

Triggers an "Accept" condition in a "Yes/No" dialog on the screen with the given index number.

Example

onAccept = "UI_DialogAcceptID(UI_GetScreenID('IPConnect'))"

Arguments

<iScreenID>: the index number of a screen.
